# 🧠 Learning System

This repo is designed to:
- Learn fast
- Remember long-term
- Revise in seconds

## 🔁 Workflow

1. Create Memory Map (1 page)
2. Create Flashcards
3. Add Code Snippets
4. Revise using triggers

## 📅 Revision Rule

- Daily → Flashcards (5 min)
- Every 3 days → Memory Map
- Weekly → Recall from memory

## 🎯 Goal

> Convert knowledge into fast recall + correct usage
